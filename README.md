# Sera Works

A modern web application for automating business workflows and processes.

## Project Overview

Sera Works is a platform that helps businesses automate their workflows and processes through a user-friendly interface. The application is built with Next.js and uses modern web technologies to provide a seamless user experience.

## Tech Stack

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- React Hook Form
- Zod for form validation
- Formspree for form submissions

## Getting Started

1. Clone the repository
2. Install dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
```

3. Start the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

4. Open [http://localhost:3000](http://localhost:3000) to view the application
